# Socrates Plan

## Purpose

Orchestrate Wonder, Reflect, Refine, and Restate to create a closed Seed.

## Success Criteria

- Runs up to 5 cycles.
- Fills goal, constraints, and acceptance_criteria.
- Writes the latest canonical Seed to top-level `## Seed`.
- Marks unresolved slots as `empty` instead of guessing.

## Boundaries

- Do not review blind spots; use `evolve-step`.
- Do not define ontology; use `ontology`.
- Do not run convergence; use `interview-harness`.
