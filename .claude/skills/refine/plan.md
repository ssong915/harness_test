# Refine Plan

## Purpose

Merge reflected meaning gaps into one user-approved shared meaning.

## Success Criteria

- Reads the active cycle's `### Reflect` section.
- Asks for one shared meaning candidate at a time.
- Writes exactly one `shared meaning: ...` line.
- Does not invent consensus without user approval.

## Boundaries

- Do not expand meanings; use `wonder`.
- Do not compare meanings; use `reflect`.
- Do not turn the shared meaning into a goal; use `restate`.
