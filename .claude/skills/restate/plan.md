# Restate Plan

## Purpose

Turn shared meaning into one executable goal sentence.

## Success Criteria

- Reads the active cycle's `### Refine` section.
- Produces exactly one `goal: ...` line.
- Confirms the goal still matches the shared meaning.
- Writes only the active cycle's `### Restate` section.

## Boundaries

- Do not expand meanings; use `wonder`.
- Do not compare meanings; use `reflect`.
- Do not create shared meaning; use `refine`.
